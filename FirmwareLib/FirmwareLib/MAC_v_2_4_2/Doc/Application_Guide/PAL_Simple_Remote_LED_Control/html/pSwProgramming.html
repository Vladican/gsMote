Firmware Programming

The pre-build firmware (binary) or the own compiled firmware can be flashed to the ATmega128RFA1 device using the AVR Studio.

The binary file (hex file) can be downloaded to the ATmega128RFA1 either from the AVR Studio project "Simple_Remote_LED_Control.aps", or opening the hex file from the AVR Studio IDE. For programming the ATmega128RFA1 Radio Controller Board needs to be connected with a JTAG ICE via the Breakout Board.

For further information about device programming see

  • AVR Studio 4
    or
  • AVR2025 IEEE 802.15.4 MAC Software Package User Guide, Section 7.3 Download an Application

For correct operation ensure that the device fuses are set to the recommended settings as shown by following figure. The fuses can be set using the AVR Studio, too.
